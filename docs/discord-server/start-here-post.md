# Discord Start Here Post

Use this as the first visible message in `#start-here`.

Keep this channel clean. It should answer three questions fast:

- What is this server?
- Where do I go first?
- How do I join or ask about a league?

## Main Post

```text
Welcome to the VBP Fantasy Network.

This server is the home base for VBP leagues, league openings, rules, announcements, and manager questions.

Start here:

1. Read rules and links
Go to #rules-and-links for the main site, league constitutions, format centers, and server expectations.

2. Find an opening
Go to the League Openings category. Each league type has its own channel with:
- a quick format explanation
- a live status board showing open/full leagues
- join links when a league is actively recruiting

3. Ask questions
Use #league-questions if you are deciding which league fits you.
Use #fantasy-football-draft-questions for draft, roster, keeper, or strategy questions.

4. Current owners
Use your league hub when one applies:
- #redraft-bracket
- #dynasty-bracket
- #best-ball-union

VBP is built around organized leagues, clear rules, public constitutions, and active commissioner communication. If something is unclear, ask before joining.

Main hub:
https://vbp-fantasy-network.vercel.app/
```

## Optional Follow-Up: How Openings Work

```text
How league openings work

Each League Openings channel has two parts:

1. Format explanation
This tells you what the league type is, how rosters work, and who the format is best for.

2. Live status board
This shows whether leagues are open, full, or pending.

Status markers:
- Green circle on the Status line = open
- Yellow circle on the Status line = coming soon or pending
- Red circle on the Status line = full or closed

Full leagues may still be listed as proof of active network history, but full leagues will not include join instructions.
```

## Optional Follow-Up: New Member Expectations

```text
New member expectations

- Read the constitution before joining a paid league.
- Ask questions before paying if anything is unclear.
- Keep league and payment conversations organized.
- Do not spam DMs or recruiting channels.
- Respect draft timers, league deadlines, and commissioner rulings.
- League integrity matters more than winning an argument.

If you are here to join a league, welcome. If you are already in a league, help keep the server useful for the next manager who lands here.
```

Methodology

The point is not to predict the standings perfectly. It is to create a repeatable preseason lens that feels fair, explainable, and fun to argue with.

55% Best legal starting lineup value
30% Top-20 roster depth value
10% Top-three quarterback room value
5% Rookie and draft momentum

Research Notes

  • The value layer uses LeagueLogs Superflex Dynasty rankings, which describe the Market Index as a model built from current ADP and weekly projections.
  • The Sleeper API supplied the league settings, rosters, manager/team labels, and rookie draft results.
  • The model uses this league's actual lineup structure: 1 QB, 2 RB, 3 WR, 1 TE, 3 FLEX, and 1 SUPER_FLEX.
  • Because the league has no current scoring yet, records, points for, and Max PF are intentionally excluded from this preseason version.

# League Notes

This folder stores commissioner-facing recruiting notes derived from the public constitutions and the current league data.

Use these notes when you want:

- quick hook points for a league
- trust signals pulled from the constitution
- a reminder of what makes one room different from another
- cleaner talking points for DMs, posts, or replies

## Files

- `league-hooks.md`
  - Main recruiting-note bank for the active leagues you are currently filling.
  - Organized by league / division so you can quickly find usable hook points without rereading full constitutions.

## Scope

These notes are not the governing rules themselves.

They are short summaries of:

- structure
- payment model
- draft pace
- scoring / lineup hooks
- recruiting angles that are worth leading with

If anything here ever conflicts with the constitution, the constitution wins.

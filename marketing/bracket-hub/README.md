# Bracket Hub Copy

Use this folder for the Sleeper-based bracket intake hub.

Files:
- `hub-ad-copy.txt`
  - Combined public-facing ad copy that sends people into the Sleeper hub.
- `pinned-messages.txt`
  - Paste-ready pinned messages for the Sleeper hub itself.

Purpose:
- Keep bracket-hub intake messaging separate from the broader recruiting copy bank.
- Keep the hub framed as one combined entry point for both `Dynasty Bracket` and `Bracket Redraft`.
- Avoid turning the hub into a rules dump before people join.
